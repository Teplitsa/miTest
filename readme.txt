Interactive test in accordance with ONWAM methodology.

== Description ==

This plugin is developed under support and supervision of [Теплица социальных технологий](http://te-st.ru/).

It allows to create interactive tests in accordance with ONWAM methodology for people
to benchmark their websites through the modern usability practices.

== Installation ==

See [Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

After activating it, please refere to internal help (MITEST->Settings->Help) for information about creating modules and questions.
